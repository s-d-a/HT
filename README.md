# HT

HT is more than a DLL extension for HaugTouch.

You get pre-configured solutions for the most DCS modules:

	A-10A
	A-10C
	BF-109-K
	F-15C
	F-86F Sabre
	FW-190D9
	KA-50
	Mi-8MTV2
	MiG-15Bis
	MiG-21Bis
	MiG-29A
	MiG-29S
	P-51D
	SU-25A
	SU-25T
	SU-27
	SU-33
	UH-1H

All these configurations are customizable.
